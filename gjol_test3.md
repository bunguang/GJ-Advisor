## <a name="overview"></a>前言

## <a name="basic"></a>一，基本问题
在此章中，我们将对古网这一测出现的一些和手感相关的基本问题进行讨论，并尝试着提出一些解决方案来作为参考。这些问题一般都不涉及到深层的职业企划以及系统设计，只是一些和表现相关的问题。然而，问题虽小，却均和直接用户体验比较相关，还望烛龙的stuff可以好好调整下。<br>
**( *Note：这里的建议只是一个脑洞，一个思路的提供，具体的solution我相信烛龙的staff一定能想到更好的方案* )**

### <a name="basic1"></a>问题一：御剑过大的动作前摇
* **问题叙述：** 这是这一测，针对御剑这个职业最为严重的问题。大部分技能，例如混沌开元，剑元始一，以及冲凝盈霄，太清御云剑，甚至就连幻剑的释放，都有着比较严重的前摇硬直。这种很大的硬直，主要造成了两个问题：一是手感上的不舒服，不跟手，不能心随意动，感觉拖泥带水，迟滞感很难受；二是在pvp，或者移动pve战等需要快速反应的环节，御剑的输出很受限制，过大的前摇导致目标常常已经移动除了攻击范围，或者在前摇过程中就被打断。<br>
然而同时，我觉得这个问题是这一测最为匪夷所思的问题。御剑的其他技能，例如云龙击，太虚证道剑等，释放速度都非常快，几乎没啥前摇；然后这一测中其他职业，例如天罡等，攻击手感也非常舒服，几乎没有迟滞感；而即使是上一测的御剑，也没有这么严重的前摇迟滞，所以真是让人感觉摸不着头脑。

*  **建议：** 没啥好说的，还是建议 **大量缩小御剑各个技能的动作前摇** ，让御剑也能拥有和天罡一样，没有啥迟滞感的手感。实际上不仅仅是御剑，**所有职业技能企划虽都有不同，但希望都能往天罡的手感方向靠拢**，让我们都能感受一下那种畅快随性的手感，而不是只能眼巴巴地看着流口水。<br>
远程dps和近战dps的输出强弱，我更倾向于从数值上进行调整，而不是从动作企划上。如果真的是为了照顾近战dps，故意把远程dps的手感改的迟滞薄弱，来换取同等平衡的输出环境的话……那我觉得是得不偿失的。

### <a name="basic2"></a>问题二：御剑的右键特殊技能循环不明
* **问题叙述：** 这是我在上一测结束后就提到过的一个问题，这一测仍然没有解决，那就是 **对于御剑而言，在进行大量普通攻击后，经常不知道特殊技能循环到哪一个了**。当然，玩家可以通过看下方的技能图标来判断次序，但是一来在紧张的战局中，花费时间去看下面的技能栏就是一个非常危险的动作。二来御剑的图标每个颜色和模式都差不多，从模式识别的角度来说，要分清还真不太容易。<br>
然而，这个问题天罡却不会遇到，我会很清楚的知道我下一段特殊攻击是什么，不需要去看技能栏，也不需要去记，这是为什么呢？这是因为，**天罡的四段普通攻击动作差异都很大，我可以通过动作，判断现在特殊技能循环到哪了**。举个例子，如果我现在看到天罡使用了第三段普通攻击，那么我就知道现在特殊技能已经循环到第四个了。

*  **建议：** 其实就是建议 **重新设计御剑的普通攻击动作，让每一段普通攻击动作都有比较大的差异**，这样玩家就可以从攻击动作上直观判断出我现在技能已经循环到第几段了。现在三测的普通攻击动作，每一段之间虽然也有差异，但差异实在太小，在战局中用余光观测还是有些困难。<br>
另外一个和普通攻击有关的问题，在这里就一道说了，那就是御剑的普通攻击技能名称叫做“玄凝剑”，但是实际上的技能动作却是“一阳指”，完全没有剑的样子出现。也常常有玩家吐槽到御剑的剑一直都背在背上，光靠手指进行技能释放，都快成“御指”了。所以 **希望之后御剑的技能，无论是普通攻击还是其他攻击，可以多一些剑在手上或飞出来的效果，多一些交互**。（顺路提一下，一二测御剑的普通攻击动作太柔太缓不美观，即使要修改也请不要还原成那个样子）

### <a name="basic3"></a>问题三：滞空时机动性太差
* **问题叙述：** 这一测新加入的浮空攻击系统确实不错，但现在这个系统的问题是当我们在浮空之后，空中的机动性非常糟糕，具体体现在两个方面：一，空中无法进行闪避操作，如果在空中正好处于某个aoe的范围圈内时，只能眼睁睁看着人物落在红圈中；二，在空中使用蓄力技能，例如御剑的冲凝盈霄，蓄力过程中无法移动或转身来改变面向（地面上是可以的）。

*  **建议：** 最直接的，**一来是加入空中的闪避动作**，并且在闪避后人物直接进入正常下落速度，而不是使用技能后的滞空状态；**二来是允许部分职业在空中蓄力时可以进行面向的改变**，来让技能的释放更具灵活性。

### <a name="basic4"></a>问题四：精准连击的显示和叠加问题
* **问题叙述：** 精准连击是这一测中新加入的系统，也就是just输入系统。其他数值上的问题暂且不论，我们这里就单纯说说精准连击在手感和反馈上的问题。其中最大的问题还是：精准连击成功的提示太不明显，在纷乱的战局中很难听到音效，以及看到自身地下圆圈的聚拢效果；同时，精准连击按照设定是有叠加层数的，叠加得越高，收益就越高，但是游戏中竟然没有相应的buff显示连击收益，实在是奇怪；最后就是，精准连击允许叠加的层数太高，容错率太低，导致大家为了精准连击放弃了其他技能，不敢中断左右键，操作压力太大。

*  **建议：** 建议分为两方面，**第一方面是优化精准连击成功后的显示效果**，在连击成功时进行全屏颤抖或者其他效果，方便玩家观测，**同时在屏幕一测进行连击计数**，显示“一次连击”，“二次连击”等字眼，方便玩家知道现在精准连击的叠加层数。<br>
第二方面，是 **限制精准连击的最高叠加层数**。也就是说无论今后精准连击系统如何修改，最高的叠加层数都不建议超过5层，**或者可以考虑提高容错率**，例如玩家有一次精准连击没有连上的话，并不直接清空连击计数，而是仅仅在连击计数上减1或者减2之类的。总而言之，就是减少玩家在操作时候的负担和压力。

### <a name="basic5"></a>问题五：星蕴定式，以及特殊技能槽的制作与切换
* **问题叙述：** 这个问题实际上和古网的设定无关，也没啥系统上的修改，只是一个人性化的问题而已。那就是，星蕴定式的设计问题。例如魔兽的双天赋系统，指的是我给你两套天赋，你可以在两套天赋之间进行切换，这听着和古网的设定很像，对吧？然而，却不是这样的。魔兽的双天赋，实际上可以两个天赋选择的是一样的专精，也就是说两个天赋可以都是同一个星蕴定式，只不过一个拿来用做pvp，一个拿来用做pve而已。<br>
这就是古网现在的问题，随着pvp和pve环境的不同，我们即使都是dps定式，在星蕴和技能选择上可能也有不同。但是现在的古网，只能允许你保存一套dps定式和一套t/奶定式。对于御剑这种t定式基本没用过的职业而言，几乎就是浪费了。每一次修改相应的星蕴，还必须回到门派洞天修改，显得太过麻烦。

*  **建议：** 首先是星蕴定式，**允许玩家保存两套，或者三套星蕴模式。其中每套星蕴模式，可以自由在两种定式方向之间选择**。拿御剑举例，御剑可以设定三套星蕴模式倒是神虹飞星篇（也就是dps定式），只不过一套星蕴是pvp专用，一套星蕴是小怪专用，一套星蕴是pve boss专用，方便玩家随时切换。<br>
其次是特殊技能的保存，现在每次更换技能都要拖选，显得太过麻烦。**希望可以提供一些快捷的选项，例如玩家可以保存三套特殊技能设计，在非战斗状态中，技能选单里可以进行方便的切换**，而免去了玩家每次都要手动拖选的麻烦。

### <a name="basic6"></a>问题六：通用技能的设计，与过于复杂的战斗操作
* **问题叙述：** 实际上，这一测的通用技能设计相比于上一测有了很大的提升，我们玩家提的建议烛龙也有采纳，那就是通用技能主要承载的是一些增益buff，解控以及回复的效果。不过嘛，问题还是有的。让我们理一理，通用技能的键位设计在F1到F4几个键位，也就是几个我们按起来不太方便，不太舒服的键位。**为了手感考虑，这样的键位设计，对应的技能应该是一些不太常用的，偶尔突发情况下才会使用的技能，对吧**？这应该是通用技能设计的基本原则，那根据这个原则，一些例如回血，减伤，解控之类的通用技能设计还是非常贴切的。<br>
然而，这一测的通用技能里，却有着一些例如短时间内大幅增大攻击力，持续回蓝，让对方易伤之类的效果存在。这样的设计，就不符合上面的原则了，**因为这样的技能并不是偶尔才会使用，而是基本上CD一结束就会再次点击**（一些灵器的设计也是这样，提升15%的会心之类的，很多时候一冷却结束就要使用）。这样的设计，无形中就会增大了dps等操作的压力。本来玩家要关心的事情就挺复杂了，boss技能，精准连击，蓄力，侵蚀条，这下还得记得定时点一点那些难按的F1到F4键，操作流畅性大打折扣。

*  **建议：** 主要的修改方向是，按照上面的原则，通用技能应该是一些偶尔突发状况下才会使用的技能，而不是一些经常会使用的技能，所以建议保留那些回血，减伤，解控的技能，而 **去掉那些大幅增大攻击力的，持续回蓝，以及让地方虚弱的技能。这些技能，实际上可以转移到奶妈职业的身上**，让她们不单单身兼回血重任，还肩负着全队的buff控制，来控制队伍的输出节奏。同理，一些灵器的主动增益技能也可以被取消掉，换成被动增益之类的效果。<br>
总而言之，**就是希望每个职业定位都可以专心做好自己的事，而不是什么都去掺一腿**。让dps可以专心钻研输出手法和争取输出环境，一些其他的问题交由奶妈来考虑，来使得每个职业的专注感和流畅感得到提升。
