# 《古剑奇谭网络版》 满级测试技能与星蕴建议整合书

## 前言
在和二测分别了大半年之后，我们又迎来了满级邀请测试，也就是传说中的2.5测。<br>
怎么说呢？2.5测还真的是，全方位的剧变。无论是在微博上，还是贴吧上，都掀起了一阵阵褒贬不一的“讨论热潮”呢XD<br>
不过对我而言，这一测下来，虽有不足，我看到的却更多是希望，是进步。于是，在开始指出不足以及提出一些自己的建议之前，我想先来罗列一下这一测中我感受到的惊艳之处，给小龙人们打打鸡血。<br>
*(Note: 因为无论是二测还是三测，我主玩的职业都是御剑，所以在下面提建议的时候基本都是以御剑来举例的，特此说明下。)*
* 画面真的是太棒了，我完全没有想过烛龙能做到这种水平……真的，**无论怎么赞美都不过分**，作为一个古粉，我很骄傲。

* 动作设计和手感进步很大，特别是天罡的动作快感……天啊，玩到手里 **简直指尖都在跳舞** 。还有御剑，抛开数值不说，这一测的御剑比上一测玩起来更加爽快了，无论是玄凝剑射出的特效，音效还是击中怪物后的反馈，还是现在自如的闪避，都十分酥爽。

* 远程现在可以 **边走动边释放技能** 了，作为一名御剑真的是深得我心，虽然这部分可能还有一点小建议要提，但是起码整个系统已经做出来了，真是不容易。

* 御剑的整体技能企划要比二测时进步很多。二测之时虽然御剑在数值上很强，但在操作中拘束太多。可是这一测的御剑，一来多了很多控制技能，以及瞬间位移技能，**完全让御剑灵动起来了**，更有飘逸的剑客之感。

* 技能的丰富度有提升，CD控制和循环更为自如，不会发生二测中那种34技能CD后，我们只能重复使用12普通攻击的囧况了。**整体战斗中技能循环更丰富了**。

* BOSS现在的技能多为非指向性的AOE技能，相比于二测的BOSS，**更加考验玩家的走位和操作**，而不是一概地站桩输出就行了，副本现在钻研起来更有乐趣了。请继续这样的副本BOSS设计思路！

OK，好话到此为止，既然是评测与建议，当然是要以指出问题为主了。<br>
既然其他玩家的评测建议文中，已经对例如UI，服装，照影，装备等问题进行了建议，本篇文章将不会再对这些内容进行赘述。可以说，此篇建议文主要是针 **技能** 和 **星蕴成长** 两部分所提出的建议，整体行文也是围绕这两点所铺开的。<br>
那么接下来，开始严肃的批判环节咯。

## 引子：关于PVE与PVP的权衡
* **问题叙述：** 这一测主要测试的一个方面便是古网的PVP系统。从技能体系中可以看出，古网之前的连招系统进行了大量的修改，来适应PVP的节奏。然而，PVP想要调整平衡？实话实说，没有那么容易。整体而言，感觉这边的性价比不太高。<br>
同时，我认为古网应该还算是一款以PVE为主的网游（类似于FF14）吧？我身边的古粉小伙伴，基本上都是忠诚的PVE党，鲜有PVP党的身影。就我个人的感觉而言，古网的目标群体中，似乎也是PVE党占到了绝大多数。

*  **建议：** 所以我这里想说的是：当PVP和PVE发生矛盾，需要权衡的时候，**请适当的以PVE为重**。我不怀疑烛龙的企划实力，我也知道烛龙如果想把PVP做好肯定也是能做到的，只不过自古鱼与熊掌不可兼得之，有时候该抉择的时候还是要抉择。

## 一，技能与连招
在此章中，我们将对古网本次的职业技能体系中的一些问题进行探讨，并试着从玩家角度给出一些反馈和建议。<br>
*(Note：这里的建议只是一个脑洞，一个思路的提供，具体的solution我相信烛龙的staff一定能想到更好的方案)*

### 问题一：连招，搓招的深度
* **问题叙述：** 除开通用技能之外，对于御剑而言，在这一测中，我们有：1个普通攻击，1个剑气（专精）技能，3个武器技能，4个特殊攻击，共9个基础技能。<br>
因为基础的起手技能多了（二测中起手技能只有1234四键的基础技能），这一测的搓招深度相比于前几测有着降低，简化了很多。**现在的连招，更加注重不同基础技能之间的自由衔接，而不是单个基础技能的深入扩展**。<br>
怎么说呢……对于PVP而言，这样的修改可能更适合快节奏的战局，毕竟在PVP之中，双方都在快速的闪避，大家也基本上只有一次出手的机会，对方就脱手了，很难挤出时间等待你慢悠悠地搓招的，大家都需要在短时间内，进行一次高输出的伤害。<br>
但是，这种修改对于PVE的体验而言，损伤真的是……巨大的。因为在PVE之中，BOSS或怪物的站位相对固定，我们还是有充足的时间来搓出我们想要的招式的。但是现在仅仅随便点击几个按键就能输出一套循环，感觉在打BOSS的时候整个人有一些……不用思考，没有乐趣。是的，乐趣，乐趣！实话实说，我是一个在玩游戏时有“多动症”的人，更喜欢能通过稍微复杂的一些操作来获得成就感~<br>
虽然说古网整体也是一个基于战法牧的框架（我对类似框架也没什么意见XD），但是我仍是希望古网能做出一些属于自己的特色。而搓招这样具有一定ACT元素的系统，在副本之时，反而能带来一些新鲜的体验呢。

*  **建议：** **增加每个武器技能的连招深度**。例如现在御剑的2号武器技能和3号武器技能，只有22,33两种连招招手法，1号技能甚至没有任何后续的连招，感觉还是乏味了些。是否可以衔接下鼠标左右键，适当扩展一下深度呢？例如可以出现“2-左-左-右”，或者“2-左-左-左-右”这样的搓招，然后在连招的过程中任意时刻在点击2便是瞬间后撤之类的。<br>
**增加每个特殊攻击的连招深度**。这一测的特殊攻击都是单次释放后没有后续连招的，可以修改为每个特殊攻击后都可以衔接鼠标左键，来形成某种后续连招。例如“右（云龙击）-左-左-左”便是在第一下云龙击之后，每点击一次鼠标左键，天空就落下一把小剑。在这个连招的过程中，任何时候再次点击鼠标右键，便立即结束当前连招，跳转到下一个特殊攻击上。

### 问题二：右键的特殊攻击循环
* **问题叙述：** 这一测的鼠标右键，承担的是多个“特殊攻击”的组合循环。和1234按键对应的“武器技能”不同，特殊攻击是无CD，自由组合，但要消耗元神的。说实话，这种将“特殊攻击”和“武器技能”拆分开来，确实在此次的测试中使得职业整体的输出手法花样多了很多。特别是对于御剑而言，当1234按键的武器技能CD之时，还可以用特殊攻击来进行过度，避免了枯燥乏味的普通攻击。<br>
但是，这样的模式却也带来了一个很大的问题，就是：**我不知道我现在右键的特殊攻击循环到哪一个了**。这次测试时候的机制是，当我使用其他武器技能或者普通攻击的时候，右键的特殊技能也会随之而循环……这样会造成的一个问题是，如果我现在使用了很多很多次普通攻击，我就很难判断出我此时右键的攻击循环到哪一个了。<br>
举个例子，当某个时候，我想着使用云龙击来AOE清理面前的小怪，却误点击到其他的技能上。在每个图标都长得差不多的情况下，真的很难用余光来判断出到底右键的技能循环到哪一个了，倒是误操作的概率非常非常大，惹得心中不快。

*  **建议：** 右键的特殊技能，修改一下循环的机制。现在的循环机制在多次普通攻击之后容易混淆特殊攻击的顺序，**可以考虑修改为其他的所有技能（包过普通攻击，通用技能，武器技能或者后续连招）都不会影响特殊攻击的技能循环**。在没有点击过特殊攻击的时候，特殊攻击的技能槽永远停留在第一个特殊技能处。**只有你点击了第一下的特殊技能，特殊技能的技能槽才会轮转到第二个技能处**。如果一定时间内不点击的话，整个特殊技能槽又会复位成第一个技能。<br>
也就是说，你想要使用第二个特殊技能，就必须先要使用第一个特殊技能。你想要使用第三个特殊技能，就必须要先使用第一，第二个特殊技能。<br>
这样的设定，能在一定程度上解决无法判断特殊技能循环到哪里的问题。虽然会使得次序靠后的特殊攻击不容易被释放（毕竟一定要先触发前两个才能释放第三个），但是实际上，特殊技能的释放顺序是由玩家自己拖选的。也就是说，玩家本身要进行选择，把常用的技能放在前面，把不常用的技能放在后面。

### 问题三：通用技能的设计
* **问题叙述：** 这一测的通用技能，游玩过后给我的最大猜想就是：烛龙制作出了很多技能，但是时间有限不知道怎么插入原有的技能系统之中，于是便粗暴地仍在了一块。特别是通用技能所对应的键位F1~F5，这明摆着就是懒得让玩家用嘛。<br>
**是啊，这通用技能，用户体验也太糟糕了**。<br>
我这里来点一点通用技能两宗罪：1，键位分配不合理，在战局紧张的时候很难抽出精力去点击F1~F5这么遥远的按键。2，通用技能里既有主动攻击（控制）技能，也有BUFF技能，感觉通用技能这一块承载的东西太多，有点冗余。

*  **建议：** 通用技能的设计需要重新企划下。首先是键位的修改，这倒是没什么好说的。其次是通用技能的效果是什么？我认为小龙人们需要思考一下。现在的通用技能中有着一些主动技能，**我认为一些主动攻击技能是可以直接放在“武器技能+普通攻击+特殊攻击”这个体系里的。而通用技能，只承担BUFF和解控的任务**。<br>
最简单的例子，便是可以把一些减速啊之类的控制技能放入特殊技能列表中，反正特殊技能是由玩家自己拖选出3个的。玩家需要减速这一类PVP技能的时候，把它拖入特殊技能槽内，不需要的时候（例如PVE的时候），拖出去便好了。同时，因为特殊技能是无CD的，所以需要我们在消耗量上对一些控制技能进行限制，例如减速这一类有控制属性的特殊技能直接清空大半个元神槽之类的……这样可以避免玩家重复使用控制技能。

## 二，星蕴，定式与残卷
在此章中，我们将对古网的整个成长系统，也就是星蕴系统进行点评，并将其定式和残卷系统放在一起讨论，提出一些自己的见解。<br>
*(Note：再重复一遍，这里的建议只是一个脑洞，一个思路的提供，具体的solution我相信烛龙的staff一定能想到更好的方案)*

### 问题一：星蕴本身太过线性
* **问题叙述：** 终于到了这里，星蕴，星蕴，这一测最大的遗憾。<br>
之前，星蕴是我这一测最期待的系统，然而看到之后却发出了疑问：这难道是烛龙的水平？<br>
不，这不应该是烛龙的水平，这不应该是那个能在动作和快感上得到长足进步，能设计出多种花式挖宝玩法的烛龙的水平。我更宁愿相信，一个更大的星蕴系统已经在筹划当中，现在这个系统只不过是一个雏形而已。<br>
好的，就是论事，星蕴系统，或者说类似的成长系统，很重要的一点是什么？我认为是，**多样性**。一个完整的系统，应该提供给玩家各种不同的选择，多到网上应该有很多不同的加点方法，加点流派在竞争……然而这一测的星蕴，有人在贴吧讨论过改如何加点吗？并没有，正如古二的星蕴系统一样，**这一测的星蕴，完全就是一条单行道**，基本上没有任何多样性可言，枯燥，乏味。<br>
如果每个人的星蕴都是一样点下去的，那这个系统还有什么意义呢？星蕴的存在，应该就是提供给玩家一个选择的机会，一个塑造自己独一无二角色的机会。

### 问题二：星蕴系统承载的内容太少
* **问题叙述：** 这一测星蕴系统存在感很低的一个原因，**便是星蕴系统本身承担的责任，被残卷和定式分走了**。星蕴不再是决定一个角色成长的全部方面，而是要依赖与定式和残卷。也就是说，星蕴系统作为一个成长系统，仅仅肩负着“属性修改”的任务，却没有“技能增强”与“技能修改”的功能（这部分功能让“残卷”系统拦去了），让我感觉有些遗憾。

### 问题三：三个系统的职能分配
* **问题叙述：** 首先来让我们聊聊定式，这个系统从二测的时候就已经存在了，通过切换定式就能切换不同职业职能。说实话，我当时的理解是因为星蕴系统还没制作完成，便先用这个定式来承载了星蕴的责任。但是这一测星蕴系统已经开放了，但是定式系统还保留着。另外一个问题是技能残卷，这个系统左右着每个技能的伤害，范围或者效果，但是这个系统却是独立于星蕴而独自存在的……<br>
**星蕴，残卷，定式三个系统交织在一起，会不会稍微显得有些繁多呢**？

### 星蕴部分整体建议
* 将残卷的功能放置到星蕴之中，让星蕴系统来承载残卷的任务。通过连接星蕴点，**使得星蕴成为一个既能影响人物属性，又能影响技能效果的系统**。从而使得星蕴这个系统，变成整个游戏的主要成长系统。

* 在上述建议的基础上扩展，将整个技能系统和星蕴系统对接起来。在玩家刚出生的是有，只有几个基础的技能，没有后续的连招。只有不断提升等级，**拿到星蕴点，才能在星蕴系统里点选，逐渐解锁新的技能或者新的后续连招，或者对招式原有的伤害范围等效果进行提升**。<br>
同时，玩家拿到星蕴点，也不一定要拿来提升技能，也可以拿来提升属性。那么这样在玩家之中便会造成一种权衡：这些宝贵的星蕴点，到底是拿来提升基础属性，还是拿来解锁技能？这种选择，我认为也是提升游戏研究乐趣的一个要素。

* 提升整个星蕴系统的复杂度和多样性，**要给玩家“鱼与熊掌不可兼得”的选择**。简而言之，就是要严格限制星蕴点的数量，而且分支要多。不要发生像古二或者古网这一测的情况：星蕴点数量非常多，而且几乎全程只有一条线，只要顺着点下去，每个人点出的结果都是一样的，根本没有研讨的必要。<br>
举个例子，想御剑的云龙击，可以分为3个强化方向，一个是提升伤害，一个是扩大AOE范围，一个是触发一定几率的减速效果（为PVP设置），其中每个方向又分为3个等级。这时候就需要玩家做出决定，因为星蕴点是极为有限的，到底是拿来强化哪个方向？还是放弃强化，转而点出云龙击的后续连招？抑或和上述建议所说的那样，把有限的星蕴点拿来提升别的属性？<br>
**有了选择，有了分支，这个系统才会有钻研的乐趣**。

* 星蕴定式由玩家自己定制，而不是采用固定的两种。现在的星蕴定式，其实说穿了就是两套心法（属性）。我们只能在这两种状态中进行切换。用数学的语言来表示，就是他们之间是离散的。那我们能不能将这个系统变的更连续化一些呢？<br>
延续着上述建议的思路，假设现在星蕴系统已经非常多样了，那么是否可以让玩家自由定制其中的路线，再把自己的路线储存成某种专精？例如，我想塑造一个输出一般，防御也一般的御剑，那么我在星蕴中便可以进行相应的点选，兼顾体和敏，平衡输出类技能分支和防御类技能分支。<br>
之后，我们可以将自己的选择存储成一种星蕴定式。我们可以通过自己的调教，自己制作出属于自己的输出定式，防御定式或者PVP定式，并且可以随时切换。<br>

* 其实上面的四个建议，只是一个理想的情况而已。例如上一个关于定式的建议，其实主要提供的只是一种思想，而不是具体的解决方案。实话实说，如果上面的所有建议都要做到的话，那么星蕴系统将会变得十分庞大，并且对于数值平衡也是一个极大的挑战。所以，**小龙人们可以对上述建议进行拆解，取舍以及精简，来达到适用的效果**。<br>
然而，无论怎么，我对于星蕴这块的成长系统和核心思想还是：1，具有多样性和分支，要让玩家“鱼与熊掌不可兼得”。2，能在星蕴中调整每个技能的效果，而不仅仅是属性提升。

## 三，其他杂项
这部分我们将给出一些小建议。这些建议不会涉及到具体系统的大修改，而是一些细节的小修小补。如果小龙人们通过完善这些细节，也许能带给玩家更舒适的游戏体验呢。

* 御剑的武器选择。当然，实体剑我是很喜欢的，甚至在我心目中 实体剑 > 剑匣+气剑。但是……能不能有另外一种选择，就是：**剑匣+实体剑** 呢？毕竟现在御剑的背后只有一把细细的剑的话显得太空了，从比例上看不太美观，背着个剑匣在视觉效果上反倒比较突出。

* 走动释放技能的时候，会有小卡顿。这一测例如御剑这样的远程攻击角色，在一边走动一遍普通攻击时，会出现一个小问题，就是在普通攻击结束后的0.5s内，人物的移动速度还会停留在普通攻击时慢走状态，导致有些操作上的不流畅。希望能改为在 **普通攻击结束后就可直接变回正常行走状态**，这样手感会舒服一些。

*  闪避CD机制的修改。这一测采用的限制无限闪避的方法是每次闪避之后都会导致一个CD。虽然这样可能在PVP中卓有成效，但是在PVE之中，常常导致玩家一次闪避的距离来不及躲过BOSS的AOE范围，所以希望能 **修改下闪避的CD机制，改为每3，4次闪避才会触发CD，或者消耗某种气力槽等**……总而言之就是希望能给玩家多次连续闪避的机会，来提高副本的操作性和容错性，同时也能略微提高一把手感。

* 既然已经和欢聚合作了，我觉得也可以考虑下 **内置YY语音的功能，在组队之后可以由队长自由选择开启**，免得玩家还需要自己外开YY，导致不必要的资源和时间损耗。我觉得这还是很能提高用户舒适性的。

* 技能描述的问题。我还是希望在技能描述的页面，**能把技能伤害的数值，冷却时间，技能效果等清清楚楚地写出来**。这样精确的叙述有助于玩家更好地理解各个技能，熟悉技能的使用与节奏。

* 副本中的复活点设置问题。希望在 **副本中的每个BOSS身前都能有一个龙星阵**。这样在开荒团团灭的时候，就不需要花费无谓的时间来从头赶路了，玩起来也更舒服。
